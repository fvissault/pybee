🏛️ Héritier de Forth

Beetle s'inspire directement du langage Forth, créé dans les années 1970 par Charles H. Moore : un langage minimaliste, modulaire et orienté pile.

Forth : un langage pionnier

En 1958, un jeune informaticien nommé Charles Moore, chargé de calculer des trajectoires de satellites, entreprend, pour faciliter son travail quotidien, la construction d'une boîte à outils sous la forme d'un interpréteur de commandes. D'année en année, cet interpréteur est amélioré et rendu indépendant de la machine hôte. En 1968, il prend le nom de Forth, et en 1970, il est suffisamment mûr pour faire l'objet d'une première publication comme « Langage pour calcul interactif ».
Forth a été conçu pour être simple, efficace et auto-suffisant. Il ne dépendait d'aucune machine particulière et permettait d'écrire aussi bien des systèmes d'exploitation que du code applicatif. 

: square ( n -- n² )
  dup *
;

Cette syntaxe concise, fondée sur la pile, a influencé de nombreux langages modernes — dont Beetle.

De Forth à Beetle

Et voici Beetle qui est donc un langage de plus haut niveau qu'à l'origine Forth puisque nous n'appliquerons que les concepts généraux de Forth. Le but de Beetle est aussi d'apporter des outils simples pour la création de page web, ce qui n'existe pas vraiment dans les versions antérieures de Forth qui est fortement axé sur la génération de code assembleur et l'automatisation. Beetle reprend cette logique séquentielle et cette philosophie minimaliste. Mais il la transpose dans le monde du web et de Python : fichiers .btl, séquences dynamiques, packages extensibles. 

<?btl
: welcome
  "Bienvenue sur Beetle !" .
;
welcome
?>

Là où Forth pilotait du matériel, Beetle pilote des applications web. Même esprit, nouvel environnement.
