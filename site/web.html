⚙️ Fait pour le Web

Beetle a été conçu pour le web moderne, avec un interpréteur Python capable de générer du HTML dynamique.

Génération HTML simplifiée

<?btl
webreset
title "Formulaire d'inscription" addcontent
html "lang" lang? addattr

( Intégration du formulaire d'inscription )
head "userarea/css/signupform.css" headlink addcontent
head "defer" "userarea/js/signupform.js" headscriptfile addcontent
body "action_page.btl" "term_privacy.btl" signupform

html @ generate cls
?>

Quelques lignes suffisent pour produire une page complète et localisée.

Intégration naturelle

Beetle s’intègre facilement avec Python et le CGI. Il peut servir à générer des pages, automatiser des envois de formulaires, ou interagir avec des bases de données.
