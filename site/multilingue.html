🌍 Multilingue natif

Beetle intègre la traduction directement dans son langage, sans bibliothèques externes.

Une approche clé/valeur simple

Il suffit de définir vos textes dans un bloc JSON multilingue. Beetle sélectionne automatiquement la bonne langue selon le contexte utilisateur. 

<btl
{
  fr : { login : "Connexion" },
  en : { login : "Login" },
  de : { login : "Anmelden" }
} var loginform-translate drop

: _login
    local key
    loginform-translate @ lang? cell@ key @ cell@ 
;
?>

Utilisation directe dans le code

<?btl
"login" _login .
?>

➡️ Le mot affiché dépend de la langue de l’utilisateur.
