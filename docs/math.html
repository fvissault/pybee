XII.2. Module 'math'

Précédent : XII.1. Package 'core' | Index | Suivant : XII.3. Package 'file'

Le package 'math' vous offre tout l'outillage nécessaire pour faire des mathématiques avec Beetle. Cela comporte la trigonométrie, la manipulation des nombres complexes, le calcul matriciel, les nombres aléatoires, etc... Vous pouvez également faire du calcul quantique.

1. Ensemble des mots

L'ensemble des mots qui composent ce package est le suivant :

2. Fonctions arithmétiques

2.1. Mot 'abs' : valeur absolue

'abs' positionne sur la pile de travail la valeur absolue d'un nombre n.

La syntaxe de ce mot est :

Syntaxe :
n - abs(n)

Exemples :
Beetle> -5 abs .cr
5
ready
Beetle> 5 abs .cr
5
ready

2.2. Mot 'cbrt' : racine cubique

'cbrt' positionne sur la pile de travail la racine cubique d'un nombre n.

La syntaxe de ce mot est :

Syntaxe :

Exemples :
Beetle> 8 cbrt .cr

ready
Beetle> 9 cbrt .cr

ready

2.4. Mot 'comb' : nombre de façon de choisir k éléments dans n éléments sans répétition ni ordre

'comb' positionne sur la pile de travail le nombre de façon de choisir k éléments dans n éléments sans répétition ni ordre.
k et n doivent être supérieurs à 0

La syntaxe de ce mot est :

Syntaxe :
n k - comb(n, k)

Exemples :
Beetle> 10 3 comb .cr
120
ready
Beetle> 3 10 comb .cr
0
ready
