I. Prérequis

Pour utiliser Beetle, un certain nombre de prérequis (qui ne sont pas si importants) sont nécessaires. Vous pouvez exécuter plusieurs actions :

  • faire fonctionner Beetle dans une page web et non seulement dans une console
  • utiliser une base de données telle que MySql

1. Apache

Pour pouvoir exécuter ces 2 fonctionnalités décrites ci-dessus, nous préconisons l'installation de WampServer 64 bits (x64) 3.3.7. Wampserver vous apportera :

  • Le serveur web Apache version 2.4.62.1
  • La base de données Mysql version 9.1.0 et MariaDb version 11.5.2

- Site pour installer Wampserver
- La procédure d'installation est un peu particulière, vous devez suivre impérativement les instructions qui sont décrites ici

2. Python

Beetle est écrit en Python version 3.12.4. Bien que nous avons essayé d'utiliser les packages standards de python, nous avons quelques spécificités qu'il faut tout de même prendre en compte.
Voici la liste des packages en question :

  • keyboard - version 0.13.5
  • legacy-cgi - version 2.6.1
  • mysql-connector-python - version 9.1.0

La commande permettant d'installer un package Python est la suivante : pip install nom_du_package

Il est conseillé d'installer Python dans un répertoire au plus haut sur votre disque. Par exemple : 'c:\python'

3. Beetle

Il est conseillé également d'installer Beetle dans le répertoire 'www' d'Apache. Dans le répertoire 'www', vous allez constater l'existence du répertoire 'pybee'. Vous pouvez renommer le répertoire 'pybee' en ce que vous voulez. Ce nouveau nouveau nom devient le nom de votre projet. L'arborescence doit impérativement être respectée et se présente de la façon suivante :

  • pybee
    • docs (La documentation que vous êtes en train de consulter)
    • packages (les classes représentant les packages de Beetle)
      • errors (les classes errors des packages)
      • help (les classes help des packages)
    • userarea (zone de dépot des fichiers Beetle)
      • tmp (répertoire de stockage des sessions coté serveur)
