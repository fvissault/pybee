XI.2. Structures du langage - Tableaux

Il y a plusieurs façons pour réaliser un tableau avec Beetle. 2 types de tableaux sont envisageables :

  • les listes indexées
  • les tables de hachage

Les listes indéxées

Comme son nom l'indique, une liste indexée est un tableau dont les valeurs sont accessible par un index. Cet index commence toujours à 0 comme dans tous les langages. Les tableax n'ont a priori pas de limites autre que la taille de votre disque dur.

Nous allons donc voir ensemble comment :

  • créer un tableau
  • ajouter des valeurs
  • supprimer des valeurs

Pour créer un tableau, Beetle vous offre plusieurs solutions :

Beetle> [ 1 2 ] dump
[1, 2]

Beetle> [ 1 , 2 ] dump ( avec une virgule )
[1, 2]

Beetle> 1 2 2 array dump
[1, 2]
