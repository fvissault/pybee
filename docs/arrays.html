XI.2. Structures du langage - Tableaux

Précédent : XI.1. Structures du langage - Manipulation de la pile | Index | Suivant : XI.3.a. Structures du langage - Définitions - Généralités

Il y a plusieurs façons pour réaliser un tableau avec Beetle. 2 types de tableaux sont envisageables :

  • les listes indexées
  • les tables de hachage

Les listes indexées

Comme son nom l'indique, une liste indexée est un tableau dont les valeurs sont accessibles par un index. Cet index commence toujours à 0 comme dans tous les langages. Les tableaux n'ont a priori pas de limites autre que la taille de votre disque dur.

Nous allons donc voir ensemble comment :

  • créer un tableau
  • ajouter des valeurs
  • supprimer des valeurs
  • modifier des valeurs
  • obtenir des valeurs
  • amplitude d'un tableau

Pour créer un tableau, Beetle vous offre plusieurs solutions :

Beetle> [ 1 2 ] dump
[1, 2]

Beetle> [ 1 , 2 ] dump ( avec une virgule )
[1, 2]

Beetle> 1 2 2 array dump
[1, 2]

Beetle> [ 1 2 ] var mytable @ . ( ce tableau peut être mis dans une variable )
[1, 2]

Beetle> [ "a" 2 ] dump
['a', 2]

Beetle> [ 2 dup ] dump ( ajout d'instructions à la création du tableau )
[2, 2]

Si vous souhaitez ajouter des valeurs à un tableau :

Beetle> [ 1 2 ] var table ( on positionne un tableau dans la variable table )

Beetle> ( syntaxe : value array -- array )

Beetle> 3 table cell+ dump
[1, 2, 3]

Beetle> "Chaine de caractères" table cell+ dump
[1, 2, 3, 'Chaine de caractères']

Pour supprimer des valeurs d'un tableau, c'est également simple :

Beetle> [ 1 2 ] var table ( on positionne un tableau dans la variable table )

Beetle> ( syntaxe : position array -- array )

Beetle> 1 table cell- dump ( rappelez-vous, une liste indéxée commence toujours par 0 )
[1]

Modifier des valeurs d'un tableau n'est pas compliqué non plus :

Beetle> [ 1 2 ] var table ( on positionne un tableau dans la variable table )

Beetle> ( syntaxe : value position array -- array )

Beetle> 3 1 table cell! dump ( rappelez-vous, une liste indéxée commence toujours par 0 )
[1, 3]

Pour obtenir des valeurs d'un tableau, il faudra procéder comme suit :

Beetle> [ 1 2 ] var table ( on positionne un tableau dans la variable table )

Beetle> ( syntaxe : array position -- valeur )

Beetle> table 1 cell@ dump ( rappelez-vous, une liste indéxée commence toujours par 0 )
2

Enfin, pour connaitre la taille d'un tableau :

Beetle> [ 1 2 ] var table ( on positionne un tableau dans la variable table )

Beetle> ( syntaxe : array -- valeur )

Beetle> table cells dump
2

Les tables de hachage

En ce qui concerne les tables de hachage, les index n'existent pas. Ce sont des clés. Les clés sont des chaines de caractères. Les tables de hachage n'ont pas limite de taille autre que celle de votre disque dur.

Nous allons donc voir ensemble comment :

  • créer une table de hachage
  • ajouter un couple (nom, valeur)
  • supprimer un couple (nom, valeur)
  • modifier un couple (nom, valeur)
  • obtenir une valeur à partir de son nom

Pour créer une table de hachage, il vous faudra écrire ce type d'instruction :

Beetle> { n1 : 1 , n2 : 2 } dump
{'n1': 1, 'n2': 2}

{ et } sont 2 mots du vocabulaire de Beetle. Il n'est pas utile d'entourer les noms des éléments de la table par des simples quotes ou double quotes. Pour séparer les noms des valeurs, il est obligatoire d'utliser le séparateur ':' (deux-points) et pour séparer les couples, il est obligatoire d'utiliser la ',' (virgule).

Pour ajouter et modifier un couple (nom, valeur) dans une table de hachage :

Beetle> { n1 : 1 , n2 : 2 } var table ( on positionne une table de hachage dans la variable table )

Beetle> ( syntaxe : name value array -- array )

Beetle> "n3" 3 table cell+ . ( ajout du couple (n3, 3) )
{'n1': 1, 'n2': 2, 'n3': 3}

Beetle> "n3" 4 table cell+ . ( modification de la valeur de n3 )
{'n1': 1, 'n2': 2, 'n3': 4}

Pour supprimer un couple (nom, valeur) d'une table de hachage, il faut procéder de la manière suivante :

Beetle> { n1 : 1 , n2 : 2 } var table ( on positionne une table de hachage dans la variable table )

Beetle> ( syntaxe : position array -- array )

Beetle> "n2" table cell- . ( on cherche à supprimer n2 )
{'n1': 1}

Beetle> "n1" table cell- . ( on cherche à supprimer n1 )
{}
