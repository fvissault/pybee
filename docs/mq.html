Le package 'math' vous offre tout l'outillage nécessaire pour faire des mathématiques avec Beetle. Cela comporte la trigonométrie, la manipulation des nombres complexes, le calcul matriciel, les nombres aléatoires, etc... Vous pouvez également faire du calcul quantique.

1. Ensemble des mots

L'ensemble des mots qui composent ce package est le suivant :
