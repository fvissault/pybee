IV. Piles de données

Précédent : III. Notation polonaise inversée (NPI) | Index | Suivant : V. Dictionnaires

Beetle repose sur la manipulation de piles LIFO (Last In, First Out) pour les données ou la pile de retour pour les données temporaires et sur un ensemble de mots de base. A partir de ces mots de base, il est possible d’enrichir le vocabulaire de mots de base en apportant ses propres mots.

Les piles sont donc :

  • La pile de travail (work stack)
  • La pile de retour (return stack)

La pile de travail est la pile la plus importante. La pile de retour est une pile vous permettant de réserver des données pendant un certain temps puis de les reprendre quand vous le jugez nécessaire dans la pile de travail. Ceci est très utile et évite trop de manipulations sur la pile de travail.

Ces 2 piles stockent les mêmes types de données. A savoir :

  • des entiers (integer)
  • des nombres décimaux (float)
  • des tableaux (array)
  • des tableaux nommés (hash table)
  • des chaînes de caractères

Le mot ' (tick) permet de déroger à cette règle et va vous autoriser à écrire des mots de votre dictionnaire sans les exécuter.
