Mots de comfort

Beetle vous offre la possibilité d'utiliser des mots de comfort. Généralement, ces mots (que nous allons passer en revue plus bas) sont des mots qui n'utilisent pas la pile de travail mais utilisent la suite de la séquence.

Load

Ce mot vous permet de charger le contenu d'un fichier btl et de l'interpréter. Le fichier btl doit être précisé sans l'extension btl.

Sa syntaxe est la suivante :

load my_file

see

Ce mot vous permet de visualiser une définition si celle-ci n'est pas native. Si la définition est native, un warning vous l'indiquera

Sa syntaxe et son résultat sont les suivants :

Beetle> see dup
: dup
0 pick
;

list

Ce mot vous permet de visualiser le contenu d'un fichier btl sans l'interpréter. Le fichier btl doit être précisé sans l'extension btl.

Sa syntaxe et son résultat sont les suivants :

Beetle> list import
import math
import file
import mysqldb
import date
packages
words
?>
